## Running the app

This app has been tested in the desktop browser (chrome, firefox) and the apple iphone simulator.

### yarn

```sh
cd react-native-pokedex
yarn install
```

#### all platforms

```sh
yarn expo
```

#### web explicitly

```sh
yarn expo start --web
```

#### tests

```sh
yarn test
```
